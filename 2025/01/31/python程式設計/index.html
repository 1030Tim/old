python3 跟 C++的差別

  1. 強制型 vs. 動態型：
  2. 解釋執行 vs. 編譯執行

強制型與動態型

C++

python

  • C++ 是編譯型語言：
    將整個程式碼翻譯成機器語言後再執行。
  • Python 是直譯型語言：
    一行行讀取程式碼並即時執行。

由於編譯型語言在執行前已經將整個程式轉換為機器語言，
因此 C++ 通常比 Python 更快速。

程式語言的運作方式
機器所理解的語言叫 機器碼，
是二進位制的（只有 0 和 1），我們稱之為低階語言。

我們所學的語言屬於高階語言，透過環境中的編譯 或解譯
轉換成中級語言，稱為組合語言，最終再編譯成機器語言。
