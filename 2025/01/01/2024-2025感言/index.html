2024 -> 2025

感言

對於高中

時光匆匆2024到2025轉瞬即逝，
記憶猶新彷彿回到2022陸興高中新生訓練
那時感嘆國中如此之快
而現今高中三年即將結束
時間是如此之快

對於2024的我

說回2024，我很慶幸不如之前那樣有空虛感
去年真的很充實，參加許多活動和踏出既有的圈子
和完成屬於自己的專案真的很開心
2023的我是自大沒信心是幼稚的
2024的我知道了人外有人天外有外星人
2024我知道了這是實力為尊的世界
2024我也知道了天才是比其他人的格外努力
才看起來毫不費力

活動專案總結

  • 台大資管營
    • 與隊友相處，我知道溝通重要性
    • 多傾聽他人想法或許可以突破思維盲點
    • 人外有人天外有外星人
    • 第一次接觸Machine Learning 知識
  • 南九校&SCIST聯合寒訓
    • 第一次接觸資訊安全陌生領域（聽不懂）
    • 加入學生程式設計讀書會
  • 斗六資訊體驗營
    • 知道資安的一些領域及手法
    • kali ping3 Ddos 操作
  • 成大資訊體驗營
    • Web開發入門 （冰川）
    • Html 與少許 CSS Github page部署
  • Aisa 亞太技術年會
    • 英文的重要性
    • AI的廣泛應用word 、 excil …
    • Copile Github 應用
    • 雲端未來會更廣泛AZURE
    • 知道很多外國開發者
  • 成大醫學與科技
    • 資工並不是單純的科系而是跨領域的起點
    • 智慧醫療是常見的延伸
  • 長治社區資安巡迴宣導
    • 回饋以及感恩的心面對身旁的長輩
    • 以攻擊者去說明資訊安全的重要
    • 長輩會很缺乏資安的問題
  • Learn Regression && 撐竿跳高
    • 待改進的點還很多
    • 讓我明白了撐竿跳高與科技結合的重要
  • 屏女自主學習發表
    • 格外努力練習才能毫不費力
    • 報告時不要怕要有氣勢
    • 臨場反應很重要
  • SCIST 演算法培訓
    • 資奧國手不是天才而是刷過上千題的人
    • 正式入門C++
    • 數學邏輯超重要
  • 屏大計算機概論先修
    • 英文超重要的考卷講義是原文
    • 已拿到學分改謝劉教授

感謝

一年的結束要感謝很多人

先感謝不斷幫助我學校、課業、生活最辛苦的
媽媽！謝謝您的幫助，讓我可以進步學習
辛苦了，願您身體健康心想事成

感謝不斷指導我不願放棄我的
貴人撐竿跳高教練
葉教練

感謝平時很照顧我的班導
有問題耐心幫我解題的
倪老師

感謝指導我特殊選才幫我寫推薦信的
歐老師

感謝之前指導我如今也一直幫助我的國文老師
杜老師

感謝協助我報考以及牽線的高一老師
倪老師

感謝身邊不斷幫助我陪伴我的貴人
學校老師們
撐竿跳高的隊友
學校的同學
支持我的人
感謝你們謝謝你們的幫助陪伴
2025願你們心想事成平安順心

期許

新的一年2025初我希望我可以順利有大學讀
可以學到更多的知識和技術
希望今年可以比去年更成熟
可以讓人依靠
希望可以以感恩的心度過一整年
希望我也可以像別人幫助我一樣幫助別人
